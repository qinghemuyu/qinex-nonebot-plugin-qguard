## 权限角色

QGuard 内置角色从低到高：

```text
普通成员 < 可信用户 < 小管理 < 群管理员 < 群主 < 超级管理员
```

权限来源优先级：

```text
QGUARD_SUPER_ADMINS > 插件角色/OneBot 群角色中更高者 > 普通成员
```

可信用户及以上不会被自动审核处罚；小管理可以警告、禁言、解禁、撤回，但不能踢人；群管理员不能处罚群主，也不能处罚同级或更高权限成员。
