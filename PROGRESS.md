# QGuard 开发进度

更新时间：2026-06-21

## 当前状态

QGuard 主体群管能力已经进入可用状态，并与 AI 问答、知识库、词云和授权登记插件完成基础联动。当前仓库重点是稳定性、文档、开源整理和真实群运行后的边界修补。

## 已完成能力

- 项目骨架、PluginMetadata、NoneBot 2 插件加载。
- SQLite 自动建表和兼容迁移。
- OneBot v11 群操作封装。
- 权限系统、超级管理员、可信用户、小管理、群管理员、群主权限分层。
- 统一插件注册中心：插件可注册命令、权限、帮助、状态和开关。
- 基础群管：帮助、状态、禁言、解禁、踢人、踢黑、警告、撤回、查用户。
- 机器人消息自动撤回：支持按指令消息、聊天消息、全部或关闭分类控制。
- 最近消息缓存：支持撤回多条、误判复盘和审计。
- 群名片管理和名片锁：手动修复、扫描、消息兜底检查、自动巡检。
- 自动审核：关键词、正则、广告检测、广告词库、刷屏检测、违规积分。
- 新人保护：新人时长、链接拦截、图片拦截。
- 入群审核：暗号、黑名单、广告理由、重复申请、欢迎消息。
- 群设置保护：群名锁、匿名锁、全体禁言、头衔设置、综合巡检。
- 长期未活跃自动清理：可配置提醒时间、提醒次数、踢出阈值和扫描间隔；达到阈值后先进入待确认清单，只有主人确认后才会踢出，也支持保留/排除成员。
- 审计日志：人工操作、自动操作、失败、跳过均记录。
- 与 QInEX 智能问答联动：AI 聊天回复默认按聊天消息分类，不被指令回执自动撤回。
- 与 QInEX AnswerBot 联动：骚扰机器人可累计群管违规积分。
- 与 QFun 联动：词云基于群消息统计，可每日定时发送。
- 与 QLicense 联动：S3/P4 设备登记、QQ 配额和授权服务内部 API 调用。

## 当前验证

- 全量单元测试：`133 passed`。
- 已覆盖群管核心、插件注册中心、知识库、智能问答、词云、自动清理和授权登记客户端。
- 开源检查已确认：提示词设计稿、`.env`、数据目录、密钥、压缩包不进入 Git 跟踪文件。
