# 排障 与 卡顿速查

## 卡顿分三段,排错顺序固定

卡顿先分清卡在哪一段:**① PC 端处理**(回报率/发送模式/CPU)→ **② 传输**(硬件 USB 线材口 / 免硬件 ADB 转发)→ **③ 手机端**(省电降频/后台/画质)。

> **大多数"卡"其实是回报率设太高 + 手机端省电,这两条先查。**

## 对号入座表

| 症状 | 最可能原因 | 先试这个 |
|---|---|---|
| 触摸点**黏、慢半拍、拖枪不跟手** | 出帧缓冲 / 回报率 | S3 板载页「出帧模式」切**丝滑**;回报率设 1000;发送模式选**异步** |
| 视角**忽快忽慢、抖** | 回报率不稳 / 加速度 | 回报率**固定 1000**;关或重调加速度曲线;P4 板载页开**稳帧** |
| **整体掉帧、电脑发烫、CPU 高** | 回报率拉太高(如 8000)反噬 | 回报率**降到 1000~2000**;关录屏/直播/杀毒实时扫描 |
| **免硬件(ADB)模式**延迟大 | TCP 经转发 | 完整版开 **USB 网络共享直连**;开**自动刷新率**;线插主板 USB3 直出 |
| 投屏画面卡(不影响触摸) | 码率/分辨率/编码/WiFi | 降分辨率+码率;编码选 **H.265**;靠近路由走 5GHz |
| 怎么调都卡 | 手机端 / 线材 / USB 口 | 见下「手机端」「线材」 |

## 上位机 / PC端 一卡一卡

用户说“最新版上位机掉帧”“PC端卡卡的”“电脑端一顿一顿”,一般先当成 **QInEX 电脑端映射软件的处理或发送节奏问题** 排查,不要直接判定为知识库范围外。

一、先把回报率降到 **1000**。如果 1000 稳了,再试 2000;一变卡就降回去。

二、发送模式选 **异步**。用户描述“慢半拍、拖枪不跟手、滑屏不顺”时,异步通常比同步更合适。

三、关闭录屏、直播、游戏滤镜、杀毒实时扫描这类吃 CPU/GPU 的东西。电脑发烫或 CPU 高时,高回报率会被放大成掉帧。

四、如果是 S3 硬件模式,去 S3 板载配置页把「出帧模式」切 **丝滑**。

五、如果是免硬件模式,打开 **USB 网络共享直连** 和 **自动刷新率**,线插主板后置 USB3 口。

六、分清是“投屏画面卡”还是“触摸控制卡”。投屏画面卡按 ScreenHub 降分辨率/码率处理;触摸控制卡按回报率、发送模式、出帧模式处理。

如果用户只说“卡”,先让他补一句:用的是 S3、免硬件还是 P4,卡的是手机画面、投屏画面,还是鼠标滑屏触摸。

## 关键认知

### 回报率不是越高越好
回报率越高,PC 每秒要处理/发送的帧越多,很多电脑 CPU 跟不上,**反而抖动、忽快忽慢、掉帧**。经验:**先用 1000**,稳了再往 2000 试;一发烫/变卡就往回降。8000 只适合高端 CPU + 真有 8000Hz 鼠标的场景。

### 发送模式 同步 / 异步
- **异步(默认推荐)**:移动帧可合并、不阻塞主循环,更跟手,极限下可能丢一两个中间帧(连续移动无感)。
- **同步**:一帧不丢但发送时阻塞,偶尔顿挫。
- **觉得卡/顿就用异步。**

### 触摸点"黏"分硬件
- **S3 硬件**:板载配置页「出帧模式」切**丝滑**(最有效)。
- **PC 端通用**:回报率 1000、发送模式异步。
- **免硬件**:打开「自动刷新率」。

### 视角忽快忽慢、像抖
多半是回报率不稳或加速度曲线导致,不是"卡":回报率固定 1000;关掉/重调加速度曲线(全 1 = 等同关闭);P4 单机版开「稳帧出帧」。

### CPU 高 / 发烫然后卡
回报率降到 1000~2000;关录屏/直播/杀毒实时扫描;不看画面时关投屏(解码吃 CPU/GPU)。

### 手机端卡(PC 怎么调都没用)
给游戏 **关后台清理、关省电、设高性能**(省电会偷偷降触控采样率);手机发烫降频会卡,注意散热、别边充边玩;游戏画质/分辨率/帧率降一档;关其他后台 app。

### 线材 / USB 口
换质量好的数据线,插**主板直出 USB 口**,别用机箱前置 / hub / 劣质延长线(最容易丢包掉帧)。

## 连不上 / 没反应

### 按键摆好游戏里没反应
先分清是"任意键都不输出"还是"只有 WASD 方向键组合时怪"。如果是前者(普通自定义键摆好游戏没动静),依次查:
1. 点了「保存配置」吗?
2. 输出模式是「触摸屏」吗?
3. 参考分辨率和手机一致吗?
4. 在游戏内、且程序是**管理员权限**运行吗?

### 同时按 WASD 组合、再按方向键没反应(摇杆机制,非 bug)
症状:按住 W+A 或 W+D 斜向走,再按另一个方向键(比如 D)没任何反应。
原因:WASD 是**一个移动摇杆 / 一个触点**,不是 4 个独立键,它把当前按住的方向键合成成**唯一一个方向**(像手柄左摇杆,同一刻只能指一个方向)。左右(A/D)抵消、前后(W/S)抵消——按住 W+A 再按 D,只是把 A 抵消、方向变正前 W,所以"D 没反应"。**这是固有逻辑、和游戏内推摇杆一致,别去查输出模式/管理员/保存,那几条治不了这个。**
要让某个键在走位时独立点屏:别用 W/A/S/D 方向键,改成单独的「自定义按键」摆到自己的触点(自定义键能和摇杆同时触发、互不抵消)。详见 04_配置面板-基础布键#移动摇杆。

### 免硬件模式连不上手机
1. cmd 跑 `adb devices` 确认能识别设备。
2. 拔插 USB 后重启程序。
3. 看「设备激活 & 状态」的具体错误提示,按提示处理。

### 切换键 ALT 在某些游戏冲突
「高级参数微调」→ 切换键改成别的,例如 `\` 或 Caps Lock。

### P4 单机版连不上配置页 / 重启死循环
多半是**固件烧错芯片版本**(eco2 / eco6),换另一个 bin 重烧(见 08_P4单机版)。

### P4 有按键显示但手机无触点
看「输入监控」:设备栏空 = 没认到键鼠(换 HS 键鼠);认到了但无触点 = **没激活**或映射问题。

## 上位机打不开 / 配置面板打不开

### QInEX 打不开或启动闪退
1. 右键 **以管理员身份运行**。
2. 确认程序是完整解压后运行,不要只把 QInEX.exe 单独拎出来。
3. 杀毒软件如果拦截或隔离文件,先恢复并加信任,再重新解压。

### 配置面板空白或打不开
1. 安装或修复 **Microsoft Edge WebView2 运行时**。
2. 确认程序目录里的配套文件完整,不要删除配置面板相关文件。
3. 仍然不行时,发启动提示截图或控制台最后几行日志,注意遮住授权码、订单号、手机号等隐私。

### 投屏「控制」模式退不出来
按 `Esc` 或你绑的操控热键。鼠标点击退不出是**防误触机制**。

### 改了投屏分辨率/码率没生效
只有**码率**是热更新;其他改动要点设置面板「应用」或重开投屏。

## 反作弊相关
- 默认扰动数值很小,几乎察觉不到。确实打不准 → 开「纯净模式」关掉所有扰动。
- **压枪反作弊最敏感**(开火恒定下拉),默认关,开启自负风险,见 06_连点与压枪。
