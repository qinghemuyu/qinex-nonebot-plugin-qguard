# 配置面板 · 基础布键

电脑版运行后会自动打开**配置面板**(内嵌窗口)。这里是你摆放键位、调参数的地方。配完一定要点**保存配置**才生效。

## 面板长什么样

- **中间画布**:手机屏幕的底图,你在上面摆触摸点。所有坐标归一化(0~1),换分辨率不用重配。
- **左侧工具栏**:添加组件、高级映射组件、键冲突检查、保存配置等。可拖动标题栏移动,右上角 `−` 折叠。
- **保存按钮变橙色** = 有未保存改动。

## 几个全局快捷键

| 键 | 作用 |
|---|---|
| ALT | 切换 鼠标模式 ↔ 视角模式(可在高级参数改) |
| F8 | 显示 / 隐藏屏幕遮罩(可在高级参数改) |
| Esc | 取消画布上的"等待放置"态 / 取消选中 |
| Delete | 删除画布上选中的组件 |

## 配置文件管理(多配置档)

- 支持多套配置档(不同游戏 / 不同手机各存一套),在面板里切换。
- **切换配置档前先保存**,否则当前改动会丢。

## 基础设置(开始配之前)

- **参考分辨率**:设成和你手机一致(关系到坐标对得准不准)。
- **输出模式**:正常映射用「触摸屏」。
- **截图**:可以截手机当前画面当底图,方便对位摆点。

## 四种基础组件

### 1. 普通按键 / 鼠标键 / 侧键
最常用:把任意键映射成屏幕上某个点的"戳一下 / 按住"。
- 添加组件 → 选「自定义按键」→ 在画布上点要触发的位置 → 绑定按键(直接按那个键就录上了)。
- 鼠标左右键、侧键、甚至**滚轮上/下**(WHEELUP / WHEELDOWN)都能当键用,绑定时直接滚一下滚轮就录上。
- **一个键可以映射多个触摸点**(一键多点)。

### 2. 移动摇杆(WASD 轮盘)
把 WASD 变成屏幕上的虚拟摇杆走位。
- 自带**拟人化**(轨迹更像真人手指 + 随机微扰动,有效规避检测),默认值即可用。
- 可逐方向微调"每个方向推多远"(半径),进阶可调每方向的形状。
- 支持**半屏移动**(自由 / 跟手摇杆,适合 MOBA:在半个屏幕任意按下都生成摇杆)。

### 3. 视角滑动区域(FPS)
鼠标移动 → 屏幕视角滑动(像端游转视角)。
- **灵敏度**:核心手感,太飘就调小。
- **加速度曲线**:鼠标快慢对应不同视角速度;全设 1 = 等同关闭加速度。
- **纯净模式**:关掉所有拟人化扰动,1:1 屏比映射,追求最精准时用。
- 视角触点空闲会自动抬起(自动回中时间),避免一直占着屏幕中心。

### 4. 鼠标轮盘(Tab 类技能位)
适合需要"按住某键 → 鼠标在一圈技能位里选"的场景(如环形技能盘)。

## 配完检查

- 左侧**键冲突检查**:扫一遍有没有同一个键被多处占用(先到先得会误触发)。
- 故意叠加的(比如右键既发触点又切层)会提示冲突,这是正常的,可忽略。
- **改完点「保存配置」**。

## 摆好却没反应?

1. 点「保存配置」了吗?
2. 输出模式是「触摸屏」吗?
3. 参考分辨率和手机一致吗?
4. 在游戏内、且程序是**管理员权限**运行吗?
