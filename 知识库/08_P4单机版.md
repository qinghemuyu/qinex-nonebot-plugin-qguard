# P4 单机版(不要电脑)

ESP32-P4 单机版:**键鼠直接插板子,板子内部完成映射后把触摸发给手机,不需要电脑**(硬件费用约 120 元)。配置不再靠 PC 程序,而是用手机/电脑浏览器连板子自带的 WiFi 配置页,或用 QInEScreen 手机 APP 配。

> 它吃**同一份 config.json**,板内映射逻辑逐项复刻 PC 上位机(视角 / WASD 摇杆 / 键位 / 鼠标轮盘 / 滚轮 / 拟人化 / 连点 / 压枪全有)。典型用法:**电脑配好键位 → 导出 json → 导进 P4 → 拔了电脑单机玩。**

## ⚠️ 先烧对固件(很重要)

ESP32-P4 有两个**互不兼容**的芯片版本,固件也分两个,**烧错会卡在重启死循环、连不上配置页**:

- `QInEX-P4-ModeA-...bin`(普通版)—— **v1.x 老芯片**(2024 批次,多数现有板子)
- `QInEX-P4-ModeA-...-v3.bin`(v3 版)—— **v3.x 新芯片**(2025 批次起,eco6+)

**怎么判断芯片**:用串口工具(115200)看开机第一行 `ESP-ROM:esp32p4-ecoN` —— `eco2` 一类用普通版,`eco6` 或更高用 v3 版。

**更省事的判断**:开机串口反复刷 `rst:…CHIP_LP_WDT_RESET` 死循环、连不上,就是版本烧错了,换另一个 bin 重烧。详细烧法见固件包里的 `烧录说明.md`。

## 5 步快速上手

1. **接线**:跳线 **H3 = HOST**;键鼠插板子 **USB-A** 口;手机插 **H2 USB-C** 口。
2. **连配置页**:板子上电,手机/电脑连 WiFi 热点 `QInEX-P4-Config`(密码 `12345678`),浏览器打开 `http://192.168.4.1/`。
3. **激活**设备(未激活不输出触摸,见下)。
4. **导入** 从 PC/服务器面板导出的 config.json。
5. 进游戏,按 **ALT** 切换视角模式;手感不对就在面板拖"灵敏度"。

### 键鼠选择(硬件限制,重要)
板子的 USB 主机受硬件限制,**优先用"高速(HS)"键鼠**(多数游戏键鼠)。部分普通办公键鼠 / 无线 dongle 是全速设备,**可能不被识别**。识别情况看面板「输入监控」。

## 板载配置页:五个标签

### 状态
- **设备信息**:固件版本、MAC、空闲内存、USB 挂载、已导入配置字节数、**激活状态**、当前 VID/PID。
- **输入监控**(调试):实时显示识别到的键鼠、按下的键、鼠标位移、帧数。排障要点:**设备栏空 = 没认到设备**(换 HS 键鼠);**有按键显示但手机无触点 = 映射或未激活问题**。

### 激活
- **未激活时不输出触摸。**
- 当前用**手动激活**:复制"设备码"交客服换激活码,填入即可。
- **在线激活暂不可用**(激活服务器未搭建),先用手动激活。"连接网络"也可用于局域网联机。

### 映射
- **导入配置**:选 config.json → 导入并重启。这是 P4 的键位来源。
- **灵敏度(实时)**:拖滑块**即时生效并保存**,无需重导/重启,调视角手感最常用。0 = 用 config 里的灵敏度。
- **屏幕旋转(实时)**:0/90/180/270 四档,即时生效。方向不对时调。
- **出帧档位**:P4 的出帧节拍,四档(默认/高档/超高/极速)。档位越高**越跟手、延迟越低**,即时生效不重启。**极速是实验档(红色按钮)**:CPU/发热更高、WiFi 高负载下可能不稳,自测后再常用;默认档最稳。P4 节拍只看这里,不受电脑端配置影响。
- **触摸校准**:接采集卡/外接屏后触点偏移时用(见下)。

### 设备
- **HID 设备信息**:改手机看到的触摸屏 VID/PID/厂商/产品名。支持编辑/随机填充/存多套套餐切换/每次开机随机。改后**需重启**重新枚举。
- **kmboxNET 网络控制**:把 P4 当 kmboxNET 盒子用(见下)。
- **vPointer 光标**:关 FPS 进光标模式,鼠标移动 → 手机出现可见光标点菜单/背包。**需手机装 vPointer app + 同网**。默认关。

### 系统
- **固件升级**:网页选官方 .bin → 上传升级,免接线。**升级中勿断电**,完成自动重启。
- **调试**:开启后串口(921600)打印键鼠原始报告,键鼠不工作时打开,把日志反馈给开发者。
- **设备控制**:立即重启 / 恢复出厂(清空全部设置)。

## kmboxNET 用法(把 P4 当网络盒子)

已适配 kmboxNET 协议的软件可直接把 P4 当盒子:在"设备"标签打开 kmboxNET 开关,软件里调 `kmNet_init(盒子IP, 端口, 盒子MAC)`,鼠标/键盘命令经网络送进板子,走完整映射后变手机触摸。盒子 MAC 在卡片显示。

| 连接方式 | 盒子 IP 填 | 说明 |
|---|---|---|
| 网口直连(推荐) | `192.168.2.188` | 网线直插 PC 与板子,能 ping 通即连通 |
| 网口接路由器 | 面板"网口直连"行显示的地址 | 网口模式切"路由器(自动)" |
| WiFi 直连 | `192.168.4.1` | PC 连板子热点后用 |
| WiFi 同路由 | 面板"WiFi 路由"行显示的地址 | 板子连了家里 WiFi、PC 同路由器时 |

> WiFi 与网口同时在线,卡片会列出所有可用地址,挑一个能 ping 通的填即可。加密:默认明文与 XXTEA 加密自动识别;开"强制加密"后只接受加密命令(防抓包),改动需重启。

## 物理回读(给第三方软件用)
键鼠插板子,板子负责把**物理键鼠状态回报**给软件,软件调 `kmNet_monitor(端口)` 读取。卡片"物理回读"行接入后变绿。回传频率:合并(默认,省流量)/ 实时(延迟最低)。

## 触摸校准
接采集卡/外接屏后画面比例变了、触点整体偏移时:填采集卡分辨率"自动计算"黑边,不对称再手调上下左右黑边(%)。即时生效并保存。

## 手机出现"幽灵触点" / 没插鼠标也像在动?
这是**视角自动回中时间**被设成了 0,板子会在屏幕中心一直按住视角触点。把它改成**大于 0**(例如 800 毫秒):空闲一会儿自动抬起,一动鼠标立刻恢复。

## P4 单机版 vs PC 上位机

| | PC 上位机 | P4 单机版 |
|---|---|---|
| 要电脑吗 | 要(常驻) | **不要**(键鼠直插板子) |
| 配置方式 | PC 配置面板 | 板上 WiFi 配置页 + 导入 json / 手机 APP |
| 映射逻辑 | 原生 | 复刻 PC,行为一致 |
| 键位来源 | 面板直接编辑 | 导入 config.json |
| 手机端 | 免 root 免 app(vPointer 除外) | 免 root 免 app(vPointer 除外) |
