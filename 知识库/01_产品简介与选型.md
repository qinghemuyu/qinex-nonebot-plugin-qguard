# 产品简介与选型

## QInEX 是什么

QInEX 把电脑或 USB 键鼠的操作,映射成安卓手机的触屏操作。用键鼠玩手游:鼠标控视角、WASD 走位、各种键位戳技能,还能开镜降灵敏、连点、压枪、投屏点对点操控。

**对手机系统完全透明**:不用 ROOT、不用给手机装 App(投屏 / vPointer 光标功能除外)。手机把它当成一台普通的 USB 触摸屏或标准触控设备。

## 几种用法(产品线)

| 用法 | 要电脑吗 | 要额外硬件吗 | 一句话 |
|---|---|---|---|
| **硬件模式(ESP32-S3 接收器)** | 要 | 要(S3 接收器,约几十元) | 电脑跑程序 → S3 接收器 → 手机。延迟最稳,最跟手。 |
| **免硬件模式(数据线 ADB)** | 要 | 不要(一根数据线) | 电脑跑程序 → 一根 USB 线 → 手机。免硬件,延迟天花板略高。 |
| **P4 单机版** | **不要** | 要(P4 板子,硬件费约 120 元) | 键鼠直接插板子,板子内部完成映射 → 手机。拔了电脑也能玩。 |
| **QInEScreen 手机 APP** | 不要 | 配合 P4 用 | 手机投屏 / 悬浮光标 / 在手机上配 P4 键位。 |

> 三种映射方式的"键位配置"是**同一份 config.json**,可以互相迁移:电脑上配好,导出,导进 P4 也能用。

## 该选哪个

- **想要最稳、最跟手,愿意接个小硬件** → 硬件模式(ESP32-S3)。
- **不想买硬件、电脑常开** → 免硬件模式(一根数据线)。
- **不想一直开电脑 / 想随身带** → P4 单机版(键鼠插板子,不用电脑)。
- **要在电脑上看手机画面 + 鼠标直接点** → 用完整版的投屏 ScreenHub;手机端投屏/光标用 QInEScreen APP。

## 电脑版:完整版 vs 免验证版

电脑程序有两个发行版,**核心触摸 / 视角 / 宏 / 反作弊功能完全相同**,区别只有三处:

1. **启动验证**:完整版首次启动需 QQ 群扫码;免验证版直接进入。
2. **投屏 ScreenHub**:仅完整版有;免验证版没有投屏开关。
3. **USB 网络共享直连**:仅完整版的 ADB 配置里有此勾选(降延迟);免验证版固定走普通转发。

两版本的 config.json 完全兼容,可无缝迁移。需要投屏 / 想用 USB 网络共享降延迟 → 完整版;只用键鼠映射 / 不想扫码 / 想极简 → 免验证版。

## 硬件版 vs 单机版怎么挑

- **延迟 / 跟手**:S3 硬件版与 P4 单机版都是硬件直出,最跟手;免硬件(ADB)是纯软件路径,延迟天花板更高。
- **便携 / 免电脑**:P4 单机版赢(不用电脑)。
- **配置便利**:电脑版直接在配置面板拖拽;P4 单机版要在电脑/服务器面板配好后导入,或用 QInEScreen 手机 APP 直接配。
