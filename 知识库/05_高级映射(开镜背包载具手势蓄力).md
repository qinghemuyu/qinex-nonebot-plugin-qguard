# 高级映射组件(开镜 / 背包 / 载具 / 手势 / 蓄力)

基础布键之外,这些组件让"一个键干更复杂的事"。先记两条地基,后面全靠它。

## 两条地基

**地基一:一个键,要么"发触点",要么"干别的"。**
- 普通映射 = 按键在屏幕某点戳一下/按住。
- 按键层、多态交互、轻触长按、手势、蓄力 = 让键干别的。
- 同一个键一般**二选一**。但有两个例外能叠加:
  - **开镜降灵敏(ADS)**:不占键,能贴在任何键上一起生效。
  - **鼠标左/右键**:能一边发自己的触点、一边再去激活按键层/触发组件。所以"右键开镜的同时还切层"做得到。

**地基二:扩展层 = "按住时临时换一套"。**
- 基础层 = 平时那套(摇杆、视角、常用键)。
- 扩展层 = 按住某个键时临时套上去的一套(能换按键,也能塞多态/轻触长按/手势/蓄力)。松开就回基础层。
- **关键**:组件放进某层,就只在那层激活时生效。"开镜时滚轮才切倍镜、平时滚轮不乱动" = 把那个手势放进开镜层。
- 摇杆 / 视角 / 鼠标轮盘 / 开镜 ADS **只能在基础层**(进扩展层会变半透明锁住,防误改)。能放进层的是:按键、多态、轻触长按、手势、蓄力。

## 组件一览

| 组件 | 用来做 |
|---|---|
| 开镜灵敏度(ADS) | 按住某键期间临时降低视角灵敏度,开镜不甩飞 |
| 多态交互 | 按一下进某种"模式"(背包/商店/地图),再按一下退出 |
| 轻触 / 长按双功能 | 同一个键,轻点一个动作、按住另一个动作 |
| 手势滑动键 | 一个键触发一段"从 A 拖到 B"的滑动(切倍镜、划菜单) |
| 按键层 | 按住激活键临时换一整套键位(载具/建造/法术栏) |
| 蓄力瞄准 | 按住起手、移动鼠标调落点、松手投出(手雷/弓) |

## 实战一:开镜全家桶 🔭
目标:按住右键 = 开镜 + 视角变稳 + 人自动停下 + 能 Q/E 歪头 + 滚轮切倍镜,松开全恢复。

1. **右键开镜变稳**:高级映射组件 → 开镜灵敏度 → 勾"启用";"按住键"绑右键(RMB);灵敏度先填 0.4(越小越稳)。
2. **右键的开镜触点**:基础层里 添加组件 → 鼠标右键,拖到游戏的开镜按钮位置。(右键特殊:一边戳触点,一边还能激活下一步的瞄准层。)
3. **建"瞄准层"实现自动停下**:高级映射组件 → 按键层 → 添加;名称"瞄准",激活键绑右键,方式选"按住";**勾"禁用走路摇杆"**(这就是开镜自动停下)。
4. **进这层配 Q/E 歪头 + 滚轮切倍镜**:点"编辑此层" → Q/E 加自定义按键点在歪头键位;滚轮切倍镜用"手势滑动键",触发键绑 WHEELUP(往上滚一下录),起点/终点设在倍镜滑条上往加倍方向拖,再加一个 WHEELDOWN 反方向。配完点"编辑基础层"回去,保存。

✅ 按住右键 = 开镜触点亮 + 视角稳 + 人停下,这时 Q/E 歪头、滚轮切倍镜;松开全恢复。

⚠️ 提醒:滚轮是"滚一下=点一下"的瞬时键,只适合手势/点触,别拿它激活按住型层或蓄力。键冲突检查会提示右键被多处占用,这套是故意叠加,忽略即可。

## 实战二:背包 / 商店 / 地图(多态交互)🎒
想要:按一下 B 开背包(鼠标变光标点装备),再按一下 B(或 Esc)关掉回打枪。

1. 高级映射组件 → 多态交互 → 添加;名称"背包",触发键绑 B,切回方式选"再按"。
2. 若开背包还要先点屏幕某按钮,勾"打开 tap"再用画布拾取位置。

> 前提:鼠标变光标点屏幕,需要你**能看到手机画面** —— 开投屏或 vPointer 才点得准,纯盲点不行。

---

## 实战三:上车换一套键(载具层)🏎️
想要:按住 V 进载具模式,WASD 改油门方向、Q/E 变别的,松开回正常。

1. 高级映射组件 → 按键层 → 加一层"载具",激活键 V,按住,勾"禁用走路摇杆"。
2. 点"编辑此层",给 WASD/Q/E 配载具该用的触点,配完回基础层。
3. 没在这层配的键仍用基础层。

---

## 实战四:一个键两用(轻触/长按)👆
想要:轻点 R = 换弹,按住 R = 丢武器。

1. 高级映射组件 → 轻触/长按 → 添加,触发键 R,阈值默认 200ms。
2. "轻触"点在换弹按钮、"长按"点在丢武器按钮。

> 小提示:配了轻触/长按的键,别再在普通映射里给同一个键配触点,会打架。

---

## 实战五:手雷 / 弓蓄力 💣
想要:按住 G 起手瞄准、移动鼠标调落点,松手扔出,蓄力时视角不乱晃。

1. 高级映射组件 → 蓄力瞄准 → 添加,触发键 G。
2. 起始触点用画布拾取点在技能/手雷键位上;灵敏度调拖动跟手程度。

## 避坑小抄

- 组件放哪层就只在那层生效。想"开镜才有"就放开镜层,想一直有就放基础层。
- 摇杆 / 视角 / 鼠标轮盘 / 开镜 ADS 只能在基础层,进扩展层会变半透明锁住。
- 滚轮是瞬时键,适合手势/点触,不适合按住型。
- 左右键能叠加(发触点 + 进层/降灵敏),别的键一般一键一用。
- 配完点**键冲突检查**扫一遍;右键那套故意叠加,提示可忽略。
- **改完点「保存配置」才生效。**
