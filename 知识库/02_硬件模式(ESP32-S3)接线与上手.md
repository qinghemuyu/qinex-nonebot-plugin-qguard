# 硬件模式(ESP32-S3 接收器)接线与上手

硬件模式是默认、最稳、最跟手的方式:**电脑跑程序 → S3 接收器 → 手机**。手机把 S3 识别成一台标准 USB 触摸屏,无需 ROOT、无需装手机 App。

## 需要的东西

- 电脑(Windows x64)+ QInEX 程序
- ESP32-S3 接收器(原生 USB 的开发板)
- USB 数据线 ×2:一根 电脑↔S3(串口),一根 S3↔手机(OTG)
- 支持 USB OTG 的安卓手机

## 接线

| 连接 | 说明 |
|---|---|
| S3 的 UART 口(TX/RX) | 连电脑,用来接收触控指令 |
| S3 的原生 USB 口 | 连手机 OTG,S3 作为 HID 触摸屏出现在手机上 |

简单记:**一根线连电脑(数据),一根线连手机(出触摸)**。两根线别插反。

## 上手步骤

1. 按接线图把 S3 接好(电脑一根、手机一根)。
2. 运行 QInEX 程序(完整版首次启动需 QQ 群扫码验证)。
3. 程序会自动打开配置页面(内嵌窗口,地址 `http://localhost:61701`)。
4. 配置面板里**工作模式选「硬件 / serial」**,设好串口号(如 `COM5`)。
5. 拖拽布键、设好参考分辨率(与手机一致)、点**保存配置**。
6. 进游戏,按 **ALT** 切换鼠标模式 / 视角模式。

## 关于 S3 接收器的「板载配置页」

S3 接收器本身带一个 WiFi 配置页(用于设置出帧模式等板子行为,与电脑端配置面板是两回事):

- **进入方式**:接收器开机时按住板上的 **BOOT 键**进入配置模式 → 手机/电脑连它的 WiFi 热点 `QInE-Touch-Config` → 浏览器打开配置页。
- **最常用的一项:出帧模式**
  - **稳定**:出帧更平滑,但有一点缓冲(略黏)。
  - **丝滑**:近乎零缓冲、最跟手,**觉得"黏、慢半拍、拖枪不跟手"就切丝滑**。

## 跟手手感不好怎么办(速查)

1. 板载配置页「出帧模式」切 **丝滑**(最有效)。
2. 电脑端「高级参数」**回报率设 1000**(不是越高越好),发送模式选 **异步**。
3. 换**质量好的数据线**,插**主板直出的 USB 口**(别用机箱前置面板 / USB hub / 劣质延长线,这些最容易丢包掉帧)。
4. 确认程序是**管理员权限**运行。

更多卡顿排查见 **10_排障与卡顿速查**。

## 编译固件(开发者向,普通用户跳过)

普通用户拿到的接收器已经烧好固件,直接用即可。需要自行烧录的,用 Arduino IDE 烧 S3 固件,关键设置:**USB CDC On Boot 选 Disabled**。(此处不展开内部实现。)
