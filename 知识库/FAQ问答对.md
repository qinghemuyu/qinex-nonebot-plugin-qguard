# QInEX 常见问题 · 一问一答

> 每条问答自包含,供问答机器人检索。

## 一、入门 / 选型

**Q：QInEX 是干什么的?**
A：把电脑或 USB 键鼠的操作映射成安卓手机的触屏操作,用键鼠玩手游 —— 鼠标控视角、WASD 走位、键位戳技能,还能开镜降灵敏、连点、压枪、投屏点对点操控。

**Q：要 ROOT 吗?要给手机装 App 吗?**
A：都不用。手机把它当成普通 USB 触摸屏/标准触控设备,完全透明。只有用"投屏"或"vPointer 光标"功能时才需要装对应的手机 App。

**Q：有几种用法,该选哪个?**
A：① 硬件模式(ESP32-S3 接收器):要电脑+小硬件,最稳最跟手。② 免硬件模式:要电脑、一根数据线、不要硬件,延迟天花板略高。③ P4 单机版:不要电脑、键鼠直插板子。④ QInEScreen 手机 APP:投屏/光标/手机配 P4。想最稳→S3;不想买硬件→免硬件;不想开电脑→P4。

**Q：完整版和免验证版有什么区别?**
A：核心触摸/视角/宏/反作弊功能完全相同。区别三处:①完整版首启需 QQ 群扫码,免验证版直接进;②投屏 ScreenHub 仅完整版有;③USB 网络共享直连仅完整版有。config.json 完全兼容,可无缝迁移。

**Q：免验证版启动没出现 QQ 扫码窗口,正常吗?**
A：正常。免验证版去掉了 QQ 验证,启动直接进配置面板。

**Q：免验证版能用投屏吗?**
A：不能。投屏/ScreenHub 相关功能在免验证版全部移除。需要"PC 看手机画面+鼠标点点点"请用完整版。

## 二、连接 / 上手

**Q：硬件模式怎么接线?**
A：S3 接收器一根 USB 线连电脑(串口/数据),另一根连手机(OTG,出触摸)。两根别插反。

**Q：免硬件模式怎么连?**
A：手机开"开发者选项+USB 调试",一根 USB 线连电脑和手机,弹窗点"允许 USB 调试"。配置面板工作模式选"免硬件/adb"。

**Q：免硬件模式连不上手机怎么办?**
A：① cmd 跑 `adb devices` 确认能识别;② 拔插 USB 重启程序;③ 看「设备激活 & 状态」的错误提示按提示处理。

**Q：怎么切换鼠标模式和视角模式?**
A：默认按 ALT 切换,可在「高级参数」改。

**Q：ALT 在某些游戏里冲突怎么办?**
A：高级参数微调里把切换键改成别的,例如 `\` 或 Caps Lock。

## 三、配置 / 布键

**Q：按键摆好游戏里没反应?**
A：依次查:①点了「保存配置」吗?②输出模式是「触摸屏」吗?③参考分辨率和手机一致吗?④在游戏内、程序是管理员权限运行吗?

**Q：换手机/换分辨率要重配吗?**
A：不用重摆点(坐标都是归一化的),把「参考分辨率」设成和手机一致即可。

**Q：一个键能映射多个触摸点吗?**
A：能。同一个键可以同时戳多个点。

**Q：滚轮能当按键用吗?**
A：能,滚轮上/下是 WHEELUP / WHEELDOWN,绑定时直接滚一下就录上。滚轮是"滚一下=点一下"的瞬时键,适合手势/点触,不适合按住型(别拿它激活按住层或蓄力)。

**Q：切换配置文件会丢改动吗?**
A：会。切换前先点「保存配置」。

**Q：键冲突检查老提示右键被多处占用?**
A：如果是"右键既发开镜触点、又切瞄准层"这类故意叠加,属于正常,忽略即可。左右键支持叠加(发触点+进层/降灵敏),其他键一般一键一用。

## 四、高级映射

**Q：怎么实现"按住右键开镜+视角变稳+自动停下"?**
A：①开镜灵敏度 ADS 绑右键、灵敏度填 0.4 左右;②基础层放一个"鼠标右键"触点点在开镜按钮;③建"瞄准层"激活键绑右键、方式按住、勾"禁用走路摇杆"(这就是自动停下)。详见 05_高级映射。

**Q：怎么做"按 B 开背包用鼠标点装备"?**
A：用"多态交互",触发键 B,切回方式"再按"。前提是能看到手机画面(开投屏或 vPointer),否则盲点点不准。

**Q：怎么做"轻点换弹、按住丢武器"?**
A：用"轻触/长按"组件,触发键同一个键,轻触点换弹、长按点丢武器,阈值默认 200ms。注意别再给同一个键配普通触点,会打架。

**Q：扩展层是什么?**
A：按住某个键时临时套上的一整套键位(载具/建造等),松开回基础层。组件放进某层就只在那层生效。摇杆/视角/鼠标轮盘/开镜 ADS 只能在基础层。

## 五、连点 / 压枪

**Q：有连点(Turbo)功能吗?**
A：有。绑触发键、点连点位置,设间隔/按下占比/抖动等。建议带抖动+落点随机更像真人。宏系统已废弃,用原生连点组件。

**Q：压枪(后坐力补偿)怎么开?**
A：找到压枪组件勾"启用"(默认关),绑开火键(通常左键),下拉强度从很小开始(0.002~0.01 起),对墙按一梭子调到压平。可设水平补偿、起始延迟、仅开镜时、抖动。

**Q：压枪会封号吗?**
A：有风险。开火时视角恒定下拉是很强的"非人特征",服务端容易识别。默认关闭,开启自行承担风险,务必配合抖动+起始延迟轻量拟人、别拉满强度。连点风险远低于压枪。

## 六、卡顿 / 延迟

**Q：触摸点黏、慢半拍、拖枪不跟手?**
A：S3 板载页「出帧模式」切"丝滑"(最有效);电脑端回报率设 1000、发送模式异步;免硬件模式开"自动刷新率"。

**Q：回报率是不是越高越好?设 8000 反而更卡?**
A：不是越高越好。太高会让 CPU 跟不上,反而抖动/掉帧/发烫。先用 1000,稳了再往 2000 试,一卡就往回降。8000 只适合高端 CPU+8000Hz 鼠标。

**Q：发送模式同步还是异步?**
A：默认异步(更跟手、不阻塞,极限下可能丢一两中间帧无感)。觉得卡/顿就用异步。同步一帧不丢但偶尔顿挫。

**Q：视角忽快忽慢、抖?**
A：多半是回报率不稳或加速度曲线。回报率固定 1000;关或重调加速度曲线(全 1=关闭加速度);P4 单机版开"稳帧"。

**Q：电脑 CPU 高、发烫然后卡?**
A：回报率降到 1000~2000;关录屏/直播/杀毒实时扫描;不看画面时关投屏。

**Q：免硬件模式特别卡延迟高?**
A：完整版开"USB 网络共享直连"(RNDIS)绕开端口转发;开"自动刷新率";线插主板后置 USB3 口,别走前置/hub/劣质延长线;要极致就上 S3/P4 硬件版。

**Q：用了 S3 还是卡?**
A：板载页出帧模式切"丝滑";换好数据线、插主板直出口;确认管理员权限运行。

**Q：手机端卡(PC 怎么调都没用)?**
A：给游戏关后台清理/关省电/设高性能(省电会偷降触控采样率);注意散热别边充边玩;游戏画质降一档;关其他后台 app。

## 七、投屏

**Q：投屏卡会拖累触摸吗?**
A：不会。投屏和触摸是两条独立通道。投屏想更流畅:降分辨率+码率、编码选 H.265、靠近路由走 5GHz。

**Q：投屏「控制」模式退不出来?**
A：按 Esc 或你在设置面板绑的操控热键。鼠标点击退不出是防误触机制。控制开关默认无热键,需自己绑。

**Q：改了投屏分辨率/码率没生效?**
A：只有码率是热更新,其他改动要点设置面板「应用」或重开投屏。

## 八、P4 单机版

**Q：P4 单机版要电脑吗?**
A：不要。键鼠直接插板子,板子内部映射后发给手机。配置用板子自带 WiFi 配置页(`http://192.168.4.1/`,热点 `QInEX-P4-Config`/`12345678`)或 QInEScreen 手机 APP。

**Q：P4 烧错固件卡在重启死循环、连不上?**
A：芯片版本烧错了。P4 有 eco2(v1.x 老芯片)和 eco6+(v3.x 新芯片)两个互不兼容版本,换另一个 bin 重烧。串口看开机第一行 `esp32p4-ecoN` 判断。

**Q：P4 键鼠插上不识别?**
A：板子 USB 主机有硬件限制,优先用"高速(HS)"游戏键鼠;部分普通办公键鼠/无线 dongle 是全速设备可能不识别。看面板「输入监控」确认。另外别插错口(键鼠插 USB-A,手机插 H2 USB-C,跳线 H3=HOST)。

**Q：P4 不输出触摸?**
A：多半是没激活(未激活不输出)。在「激活」标签复制设备码交客服换激活码填入。在线激活暂不可用,用手动激活。

**Q：P4 怎么调灵敏度/旋转?要重导配置吗?**
A：不用。映射页的"灵敏度(实时)""屏幕旋转(实时)""出帧档位"都是即时生效并保存,不重启。只有"导入配置"和改 HID 设备信息需要重启。

**Q：P4 手机出现幽灵触点、没插鼠标也像在动?**
A：视角自动回中时间设成了 0,板子在屏幕中心一直按住视角触点。把它改成大于 0(如 800ms),空闲自动抬起,一动鼠标立刻恢复。

**Q：P4 能当 kmboxNET 盒子吗?**
A：能。「设备」标签开 kmboxNET,软件里 `kmNet_init(盒子IP, 端口, 盒子MAC)`。网口直连填 `192.168.2.188`,WiFi 直连填 `192.168.4.1`,挑能 ping 通的填。默认明文/加密自动识别。

## 九、其他

**Q：反作弊扰动会影响精度吗?**
A：默认数值很小几乎察觉不到。确实打不准就开「纯净模式」关掉所有扰动。

**Q：手柄映射,多个手柄键能映到同一个键盘键吗?**
A：不能,同一键盘键只能映一个手柄键。但同一键盘键可以映多个触摸点。

**Q：接采集卡/外接屏后触点偏了?**
A：用「触摸校准」:填采集卡分辨率"自动计算"黑边,不对称再手调上下左右黑边(%)。即时生效。

**Q：vPointer 光标是干嘛的?**
A：在手机上显示一个可见光标,配合鼠标模式点菜单/背包时看得见点在哪。需手机装 vPointer/QInEScreen app 且与设备同网,默认关。

**Q：怎么联系作者 / 支持?**
A：完整版首启的 QQ 群即官方支持渠道;遇到键鼠不识别等问题可开「调试」抓串口日志反馈给开发者。
