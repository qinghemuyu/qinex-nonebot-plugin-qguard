# QInEX 常见问题 · 一问一答

> 每条问答自包含,供问答机器人检索。

## 一、入门 / 选型

**Q：QInEX 是干什么的?**
A：把电脑或 USB 键鼠的操作映射成安卓手机的触屏操作,用键鼠玩手游 —— 鼠标控视角、WASD 走位、键位戳技能,还能开镜降灵敏、连点、压枪、投屏点对点操控。

**Q：要 ROOT 吗?要给手机装 App 吗?**
A：都不用。手机把它当成普通 USB 触摸屏/标准触控设备,完全透明。只有用"投屏"或"vPointer 光标"功能时才需要装对应的手机 App。

**Q：有几种用法,该选哪个?**
A：① S3 硬件模式:电脑跑 QInEX+S3 接收器,稳。② 免硬件 ADB:电脑+一根数据线,不要板子。③ P4 单机模式:不要电脑,键鼠直插 P4。④ P4 上位机模式:电脑跑 QInEX,P4 当高速下位机触摸桥。⑤ QInEScreen:投屏/光标/手机配 P4。想免电脑选 P4 单机;想电脑上位机完整配置选 S3 或 P4 上位机。

**Q：完整版和免验证版有什么区别?**
A：核心触摸/视角/宏/反作弊功能完全相同。区别三处:①完整版首启需 QQ 群扫码,免验证版直接进;②投屏 ScreenHub 仅完整版有;③USB 网络共享直连仅完整版有。config.json 完全兼容,可无缝迁移。

**Q：免验证版启动没出现 QQ 扫码窗口,正常吗?**
A：正常。免验证版去掉了 QQ 验证,启动直接进配置面板。

**Q：免验证版能用投屏吗?**
A：不能。投屏/ScreenHub 相关功能在免验证版全部移除。需要"PC 看手机画面+鼠标点点点"请用完整版。

## 二、连接 / 上手

**Q：硬件模式怎么接线?**
A：S3 接收器一根 USB 线连电脑(串口/数据),另一根连手机(OTG,出触摸)。两根别插反。

**Q：免硬件模式怎么连?**
A：手机开"开发者选项+USB 调试",一根 USB 线连电脑和手机,弹窗点"允许 USB 调试"。配置面板工作模式选"免硬件/adb"。

**Q：免硬件模式连不上手机怎么办?**
A：① cmd 跑 `adb devices` 确认能识别;② 拔插 USB 重启程序;③ 看「设备激活 & 状态」的错误提示按提示处理。

**Q：怎么切换鼠标模式和视角模式?**
A：默认按 ALT 切换,可在「高级参数」改。

**Q：ALT 在某些游戏里冲突怎么办?**
A：高级参数微调里把切换键改成别的,例如 `\` 或 Caps Lock。

## 三、配置 / 布键

**Q：按键摆好游戏里没反应?**
A：依次查:①点了「保存配置」吗?②输出模式是「触摸屏」吗?③参考分辨率和手机一致吗?④在游戏内、程序是管理员权限运行吗?(注意:如果"没反应"的是 W/A/S/D 走位方向键,那多半不是这里的问题,见下一条摇杆机制。)

**Q：校准映射之后,部分映射按键失效或点不准?**
A：这通常不是硬件坏,而是校准后坐标变换、参考分辨率、黑边比例或按键层/键冲突的问题。先看是"所有键都没输出"还是"只有部分键失效":所有键都没输出才查保存配置、输出模式、管理员权限;只有部分键失效时,先确认参考分辨率和手机一致,触摸校准/黑边是否对准,再把失效触点拖到游戏按钮中心偏内一点测试。P4/采集卡/外接屏场景尤其要检查采集卡分辨率和上下左右黑边(%),边缘或某一侧失效很像黑边/比例没调准。最后点保存配置,再跑键冲突检查,确认失效按键没有被按键层、轻触长按、多态交互等组件占用。

**Q：同时按 WASD 组合(比如 W+A、W+D),再按其他方向键没反应?按住斜向再按一个方向键不触发?**
A：这不是 bug,是摇杆机制。WASD 是**一个移动摇杆 / 一个触点**,不是 4 个独立按键,它把你当前按住的方向键合成成**唯一一个方向**(像真实手柄左摇杆,同一刻只能指一个方向)。左右(A/D)互相抵消、前后(W/S)互相抵消:按住 W+A 是左前斜向,这时再按 D 只是把 A 抵消、方向变成正前 W,所以"按 D 没反应",和游戏里推摇杆表现一致。**不用查输出模式/管理员/保存。** 如果你想让某个键在走位时独立点屏,别用 W/A/S/D 方向键,改成单独的「自定义按键」摆到它自己的触点(自定义按键能和摇杆同时触发、互不抵消)。

**Q：开镜灵敏度 ADS 开了,为什么右键不开镜?**
A：ADS 只负责"按住某键时降低视角灵敏度",它不会自动点游戏里的开镜按钮。右键开镜通常要配两件事:右键自定义按键点在开镜按钮上,再让 ADS 也绑定右键降灵敏。

**Q：轻触/长按老是误触,或者和普通按键打架?**
A：轻触/长按本身就是一个键两种动作,同一个键不要再配普通自定义触点。误触多就调长按阈值,并确认轻触点、长按点都拾取对了。

**Q：滚轮切倍镜应该用鼠标轮盘还是手势?**
A：滚一下切倍镜用手势滑动键,分别给 WHEELUP / WHEELDOWN 配两个相反方向手势。鼠标轮盘组件是"按住某键进环形盘滑选",不是物理滚轮上下滚动本身。

**Q：按键层里为什么不能改摇杆/视角?**
A：摇杆、视角、鼠标轮盘、ADS 是基础层专属。按键层是临时换一套可分层的键位,适合放自定义按键、多态、轻触长按、手势、蓄力等。

**Q：蓄力瞄准和普通长按有什么区别?**
A：蓄力瞄准是按住后触点跟着鼠标拖动瞄准,松手抬起投出;普通长按只是保持一个触点。手雷、弓这类用蓄力瞄准更合适。

**Q：换手机/换分辨率要重配吗?**
A：不用重摆点(坐标都是归一化的),把「参考分辨率」设成和手机一致即可。

**Q：一个键能映射多个触摸点吗?**
A：能。同一个键可以同时戳多个点。

**Q：滚轮能当按键用吗?**
A：能,滚轮上/下是 WHEELUP / WHEELDOWN,绑定时直接滚一下就录上。滚轮是"滚一下=点一下"的瞬时键,适合手势/点触,不适合按住型(别拿它激活按住层或蓄力)。

**Q：切换配置文件会丢改动吗?**
A：会。切换前先点「保存配置」。

**Q：键冲突检查老提示右键被多处占用?**
A：如果是"右键既发开镜触点、又切瞄准层"这类故意叠加,属于正常,忽略即可。左右键支持叠加(发触点+进层/降灵敏),其他键一般一键一用。

## 四、高级映射

**Q：怎么实现"按住右键开镜+视角变稳+自动停下"?**
A：①开镜灵敏度 ADS 绑右键、灵敏度填 0.4 左右;②基础层放一个"鼠标右键"触点点在开镜按钮;③建"瞄准层"激活键绑右键、方式按住、勾"禁用走路摇杆"(这就是自动停下)。详见 05_高级映射。

**Q：怎么做"按 B 开背包用鼠标点装备"?**
A：用"多态交互",触发键 B,切回方式"再按"。前提是能看到手机画面(开投屏或 vPointer),否则盲点点不准。

**Q：怎么做"轻点换弹、按住丢武器"?**
A：用"轻触/长按"组件,触发键同一个键,轻触点换弹、长按点丢武器,阈值默认 200ms。注意别再给同一个键配普通触点,会打架。

**Q：扩展层是什么?**
A：按住某个键时临时套上的一整套键位(载具/建造等),松开回基础层。组件放进某层就只在那层生效。摇杆/视角/鼠标轮盘/开镜 ADS 只能在基础层。

## 五、连点 / 压枪

**Q：有连点(Turbo)功能吗?**
A：有。绑触发键、点连点位置,设间隔/按下占比/抖动等。建议带抖动+落点随机更像真人。宏系统已废弃,用原生连点组件。

**Q：压枪(后坐力补偿)怎么开?**
A：找到压枪组件勾"启用"(默认关),绑开火键(通常左键),下拉强度从很小开始(0.002~0.01 起),对墙按一梭子调到压平。可设水平补偿、起始延迟、仅开镜时、抖动。

**Q：压枪会封号吗?**
A：有风险。开火时视角恒定下拉是很强的"非人特征",服务端容易识别。默认关闭,开启自行承担风险,务必配合抖动+起始延迟轻量拟人、别拉满强度。连点风险远低于压枪。

## 六、卡顿 / 延迟

**Q：触摸点黏、慢半拍、拖枪不跟手?**
A：S3 板载页「出帧模式」切"丝滑"(最有效);电脑端回报率设 1000、发送模式异步;免硬件模式开"自动刷新率"。

**Q：回报率是不是越高越好?设 8000 反而更卡?**
A：不是越高越好。太高会让 CPU 跟不上,反而抖动/掉帧/发烫。先用 1000,稳了再往 2000 试,一卡就往回降。8000 只适合高端 CPU+8000Hz 鼠标。

**Q：发送模式同步还是异步?**
A：默认异步(更跟手、不阻塞,极限下可能丢一两中间帧无感)。觉得卡/顿就用异步。同步一帧不丢但偶尔顿挫。

**Q：视角忽快忽慢、抖?**
A：多半是回报率不稳或加速度曲线。回报率固定 1000;关或重调加速度曲线(全 1=关闭加速度);P4 单机模式开"稳帧"。如果是 P4 上位机模式,先把 8000Hz 降到 1000~2000 对比。

**Q：电脑 CPU 高、发烫然后卡?**
A：回报率降到 1000~2000;关录屏/直播/杀毒实时扫描;不看画面时关投屏。

**Q：免硬件模式特别卡延迟高?**
A：完整版开"USB 网络共享直连"(RNDIS)绕开端口转发;开"自动刷新率";线插主板后置 USB3 口,别走前置/hub/劣质延长线;要极致就上 S3/P4 硬件版。

**Q：用了 S3 还是卡?**
A：板载页出帧模式切"丝滑";换好数据线、插主板直出口;确认管理员权限运行。

**Q：手机端卡(PC 怎么调都没用)?**
A：给游戏关后台清理/关省电/设高性能(省电会偷降触控采样率);注意散热别边充边玩;游戏画质降一档;关其他后台 app。

## 七、投屏

**Q：投屏卡会拖累触摸吗?**
A：不会。投屏和触摸是两条独立通道。投屏想更流畅:降分辨率+码率、编码选 H.265、靠近路由走 5GHz。

**Q：投屏「控制」模式退不出来?**
A：按 Esc 或你在设置面板绑的操控热键。鼠标点击退不出是防误触机制。控制开关默认无热键,需自己绑。

**Q：改了投屏分辨率/码率没生效?**
A：只有码率是热更新,其他改动要点设置面板「应用」或重开投屏。

## 八、P4 双模式

**Q：P4 到底是单机版还是上位机模式?**
A：两个都有。P4 单机模式/Mode A 是键鼠直插 P4,板子内部映射,不需要电脑常驻。P4 上位机模式/Mode B 是电脑跑 QInEX 上位机,P4 当下位机触摸桥,更像 S3 硬件模式的 P4 高速版。问 P4 问题时先分清这两个模式。

**Q：P4 单机模式要电脑吗?**
A：玩的时候不要。键鼠直接插 P4,板子内部映射后发给手机。配置用板子自带 WiFi 配置页(`http://192.168.4.1/`,热点 `QInEX-P4-Config`/`12345678`)或 QInEScreen 手机 APP。也可以先在电脑/配置网站配好 `config.json`,再导入 P4。

**Q：P4 上位机模式要电脑吗?**
A：要。电脑运行 QInEX 上位机负责映射,P4 只负责把电脑发来的触摸帧输出给手机。上位机里工作模式选"硬件/serial",选对 P4 的 COM 口。它不是键鼠直插板子玩的那套单机流程。

**Q：P4 上位机模式和 S3 硬件模式有什么关系?**
A：使用口径很像:都是电脑上位机 → 板子 → 手机触摸。区别是 P4 Mode B 用 USB HS 做触摸设备,设备侧可以支持更高出帧,上位机会识别到 `ESP32-P4 / MODE=B`。用户排障时仍先看 COM 口、硬件/serial、上位机是否保存配置。

**Q：P4 单机和 P4 上位机怎么切换?**
A：通过 P4 配置页上传官方 OTA 包切换。单机 Mode A 想切上位机,在单机配置页上传 Mode B OTA 包;上位机 Mode B 想切回单机,在 Mode B 配置页上传 Mode A OTA 包。注意 eco2/v1.x 和 eco6+/v3.x 固件别混用。

**Q：P4 烧错固件卡在重启死循环、连不上?**
A：优先怀疑芯片版本或模式固件不匹配。P4 有 eco2/v1.x 和 eco6+/v3.x 两类固件,Mode A/Mode B 也各有 OTA 包。拿官方对应版本包处理;普通用户不要乱刷来源不明的整片镜像。

**Q：P4 的 config.json 能不能通用?**
A：主要映射配置通用。电脑端/配置网站配好的 `config.json` 可以导入 P4 单机模式。P4 单机会读取映射相关字段,忽略 PC 专属字段。P4 上位机模式则直接由电脑 QInEX 读取当前配置并实时发送给 P4。

**Q：P4 单机模式键鼠插上不识别?**
A：看板载页「输入监控」。设备栏空就是没识别到键鼠,优先换高速游戏键鼠或有线键鼠;部分办公键鼠、无线 dongle、低速/全速设备可能不兼容。再确认键鼠插 USB-A、手机插 H2 USB-C、跳线 H3=HOST。

**Q：P4 不输出触摸?**
A：先分模式。P4 单机模式里,未激活不会输出触摸;还要查是否导入配置、导入后是否重启、手机是否识别 USB 触摸屏。P4 上位机模式里,先查 Mode B 固件、COM 口、上位机是否选硬件/serial、串口有没有被别的软件占用。

**Q：P4 激活用哪个 MAC?**
A：一定用 P4 板子专属配置页显示的设备码/MAC。不要用电脑网卡、路由器列表、手机蓝牙、手机 WiFi 里的 MAC,否则很容易授权和实际板子对不上。

**Q：P4 单机怎么调灵敏度/旋转?要重导配置吗?**
A：不用。单机配置页的"灵敏度(实时)""屏幕旋转(实时)""出帧档位""稳定出帧"都是即时生效并保存。只有导入配置、改 HID 设备信息、切固件模式这类才需要重启。

**Q：P4 上位机 8000Hz 怎么用?**
A：先确认 P4 已经是 Mode B,上位机识别到 P4 后再在电脑端调轮询率。8000Hz 不是越高越好,电脑 CPU、鼠标回报率、串口发送都要跟得上。普通机器先从 1000 或 2000 试,一旦发烫、掉帧、忽快忽慢就降回去。

**Q：P4 手机出现幽灵触点、没插鼠标也像在动?**
A：单机模式里常见原因是视角自动回中时间设成了 0,板子在屏幕中心一直按住视角触点。把它改成大于 0(如 800ms),空闲自动抬起,一动鼠标立刻恢复。

**Q：P4 能当 kmboxNET 盒子吗?**
A：P4 单机固件里支持 kmboxNET。「设备」标签开 kmboxNET,软件里 `kmNet_init(盒子IP, 端口, 盒子MAC)`。网口直连填 `192.168.2.188`,WiFi 直连填 `192.168.4.1`,挑能 ping 通的填。默认明文/加密自动识别。

## 九、智能客服 / 售后闭环

**Q：为什么我只说“卡”“没反应”,机器人不直接给答案,反而先问我 S3/ADB/P4?**
A：因为“卡”和“没反应”太泛了,可能是滑屏卡、投屏画面卡、按键响应慢、全部不输出、部分按键失效、校准偏移等完全不同问题。机器人先问 S3/免硬件 ADB/P4、卡的是哪一段、是全部还是部分,是为了避免乱猜和套错排查路线。

**Q：怎么让机器人连续跟进同一个问题?**
A：同一个群、同一个用户短时间内直接说“还是不行”“没效果”“还卡”即可,机器人会结合上一轮继续排查。如果要换新问题,说“新问题”“换个问题”或“不是这个”。

**Q：问题解决后要不要告诉机器人?**
A：建议说“好了”“解决了”“可以了”。机器人会把这次路线记为已解决,以后类似问题会更偏向这条有效排查思路。

**Q：机器人一直解决不了会怎样?**
A：同一问题连续多轮仍没解决时,机器人会把最初问题、最近补充、命中的知识和最新回复整理后私信给主人。默认超过 10 轮才升级,并且只通知一次。

**Q：主人收到“智能问答未命中”后怎么补知识?**
A：直接用私信里带的命令格式: `/客服 补知识 N000001 答案内容`。答案只写公开使用步骤,不要写授权码、密钥、算法、源码或绕过方式。补完后用 `/知识 搜索 关键词` 验证能不能命中。

**Q：/客服 缺口 是干什么的?**
A：它是智能问答缺口看板,会按未解决次数、未命中次数、出现次数和已解决次数给问题排序。优先处理排在最前、且带“下一步:/客服 补知识 Nxxxxxx”的问题。

**Q：本群为什么明明补了知识,机器人还是说知识范围里没有?**
A：先看 `/知识 范围`。如果本群只启用了少数分类或 skills,新补的 FAQ 可能不在当前范围里。可以临时切全部范围,或把相关 skill 加进去,例如智能客服相关是 `qinex_ai_support`。

## 十、其他

**Q：反作弊扰动会影响精度吗?**
A：默认数值很小几乎察觉不到。确实打不准就开「纯净模式」关掉所有扰动。

**Q：手柄映射,多个手柄键能映到同一个键盘键吗?**
A：不能,同一键盘键只能映一个手柄键。但同一键盘键可以映多个触摸点。

**Q：接采集卡/外接屏后触点偏了?**
A：用「触摸校准」:填采集卡分辨率"自动计算"黑边,不对称再手调上下左右黑边(%)。即时生效。

**Q：vPointer 光标是干嘛的?**
A：在手机上显示一个可见光标,配合鼠标模式点菜单/背包时看得见点在哪。需手机装 vPointer/QInEScreen app 且与设备同网,默认关。

**Q：怎么联系作者 / 支持?**
A：完整版首启的 QQ 群即官方支持渠道;遇到键鼠不识别等问题可开「调试」抓串口日志反馈给开发者。
