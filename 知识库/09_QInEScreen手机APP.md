# QInEScreen 手机 APP(投屏 / 光标 / 手机配 P4)

QInEScreen 是一个安卓 APP,三件事:**投屏**、**悬浮光标(vPointer)**、**在手机上配 P4 单机模式键位**。不用电脑。

## 三大功能

### 1. 投屏
把手机画面投出去(给 PC 看 / 推流),支持多种传输方式。

### 2. 悬浮光标(vPointer)
在手机屏幕上显示一个**可见光标**,配合"鼠标模式"用 —— 这样用鼠标点手机菜单/背包时**看得见点在哪**,不用盲点。
- 电脑版、P4 单机模式都能驱动这个光标(需手机装本 APP + 与设备同网)。
- 默认关闭,不用就不装。

### 3. P4 映射配置(手机直接配 P4 单机模式,免电脑)
APP 内置一个**原生触屏编辑器**(不是网页),手机连 P4 热点就能把键位 / 摇杆 / 视角 / 高级组件全配好,一键推送到板子。这里主要服务 **P4 单机模式 / Mode A**。如果用户跑的是 **P4 上位机模式 / Mode B**,复杂布键仍在电脑 QInEX 上位机里做。

## 用手机配 P4 的步骤

1. 手机装 QInEScreen APP → 主页点「**P4 映射配置**」卡片。
2. **首次验证**:连 P4 热点 `QInEX-P4-Config`(密码 `12345678`)→ 点"验证"。已激活的 P4 自动通过,本机记住、之后不再验证(未激活会提示先到板载面板激活)。
3. 进编辑器,底部 4 个标签:
   - **布局**:画布上拖拽摆点,悬浮组件坞把组件拖到画布落点。手势:**拖动 = 移位,轻点 = 弹属性卡**,双指捏圆形组件改半径,空白双指缩放画布,长按出菜单。手机没键盘,绑键用触屏键名网格选。
   - **组件**:多态 / 轻触长按 / 手势 / 蓄力 / 按键层 / 连点 / 压枪 / ADS / 滚轮 / 反作弊 / 基础设置 + 键冲突检查(规则同电脑版)。
   - **配置档**:多档增删改 + 推送/拉取 + 导入导出;底部显示与 P4 的连接状态(在线/运行/内存/板子地址)。
   - **设备**:加载板子自带的配置页(实时调参 —— 灵敏度/出帧/旋转/触摸校准 —— 直接在这里调,免重启)。如果板子切到 Mode B,设备页会显示上位机模式的设备配置能力,不是手机布键入口。
4. 配好 → 推送到板子(**重启后生效**)。

> 推送的是"映射键位",**重启才生效**;灵敏度/出帧/旋转这类实时调参在"设备"标签里调,免重启。

## 注意点

- **配置安全**:APP 只改 P4 用到的字段,电脑专属字段(宏/手柄/授权/投屏等)原样保留,手机改键位不会洗掉电脑端字段。
- **网络**:连 P4 热点时手机没有互联网,APP 会把流量绑在 WiFi 上,避免安卓甩到蜂窝数据导致连不上板子。
- 构建/安装由作者提供安装包,普通用户直接装 APK 即可。
