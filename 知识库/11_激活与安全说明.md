# 激活与安全说明

## 适用范围

这篇只回答 QInEX 的表层激活和安全提醒，适用于用户问“S3 板子怎么激活”“P4 怎么激活”“激活入口在哪”“激活后为什么没反应”这类问题。

这里不包含授权算法、密钥、生成规则、绕过方法、订单查询、换绑处理、退款处理。遇到这些问题，只能让用户私聊主人或客服处理。

## S3 硬件板子怎么激活

一、先把 S3 板子按硬件模式接好，确认电脑已经能识别板子，QInEX 能正常打开。

二、打开 QInEX 配置面板，找到激活或授权相关入口。

三、按页面提示填写激活信息。不要在群里发送完整授权码、注册码、订单号、密钥或带隐私的截图。

四、激活后重新打开 QInEX，重新连接板子，再检查映射输出是否生效。

五、如果激活后仍然没有触摸输出，先确认当前是不是硬件模式、配置有没有保存、游戏窗口/投屏窗口是否选对。

## P4 怎么激活

P4 先分模式:

- **P4 单机模式 / Mode A**:在 P4 板载配置页或 QInEScreen 的 P4 配置流程里看激活状态。
- **P4 上位机模式 / Mode B**:在 Mode B 板载配置页里看设备码和授权状态;电脑上位机只负责映射和串口连接。

无论哪种 P4,激活用的 MAC / 设备码都必须从**板子专属配置页面**复制。不要用电脑网卡、路由器列表、手机蓝牙、手机 WiFi 里的 MAC,那些不是 P4 授权用的设备码。

激活后如果没有触摸输出,先按 P4 文档排查:当前是单机还是上位机模式,手机是否识别触摸,P4 是否在正确模式,配置是否已经保存/导入/下发。

如果页面提示激活失败、授权异常、订单异常、需要换绑或需要查询账号，不要在群里贴完整信息，让用户私聊主人或客服处理。

## 激活问题的安全回复口径

可以回答：

一、激活入口大概在哪里。

二、激活后需要重启或重新连接。

三、激活后没输出时先检查模式、保存配置、窗口选择、触摸输出。P4 要额外确认是单机 Mode A 还是上位机 Mode B。

四、提醒用户不要在群里发完整授权码、订单号、密钥、手机号、付款截图。

不能回答：

一、授权码生成、破解、绕过、算法、密钥。

二、订单查询、退款、换绑的具体处理。

三、让用户在群里公开完整授权信息。

## 仍然不行时补充

如果用户说“激活了还是不行”，优先让用户补充三项：使用的是 S3、P4 单机还是 P4 上位机，QInEX 当前模式是什么，激活页面有没有报错。仍然涉及订单、换绑、退款或授权异常时，让用户私聊主人处理。
