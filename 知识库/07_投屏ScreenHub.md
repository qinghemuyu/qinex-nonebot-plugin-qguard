# 投屏 ScreenHub(完整版)

把手机画面镜像到电脑,**所见即所点** —— 鼠标点哪、手机就触发哪。适合在电脑上直接操作真机界面(点菜单、背包、看大地图)。

> 仅**完整版**有此功能,免验证版没有投屏开关。投屏与触摸是**两条独立通道**,投屏卡不影响点击/视角的及时性。

## 开启

运行程序后,点 Console UI 里(心形语言按钮旁边)的投屏开关 → 弹出 **ScreenHub 窗口**。

也可在投屏配置里设成"启动时自动打开"。投屏用**独立配置文件 mirror.json**(和主配置 config.json 互不影响)。

## 窗口特性

- **画面贴合窗口**:无黑边、锁定宽高比,最大不超屏幕 90%,首帧自动居中。
- **悬浮工具条**:可停靠四边(上/下/左/右循环切换),鼠标靠近唤出、移开自动隐藏。
- **拖工具条空白处 = 移动整个窗口**;拖窗口边缘 = 缩放。
- 按钮都是系统矢量图标。

## 工具条按钮 / 快捷键

| 操作 | 快捷键 |
|---|---|
| 设置面板 | S |
| 全屏 | F |
| 窗口置顶 | T |
| 请求关键帧(画面花了刷一下) | K |
| 切换停靠边 | D |
| 加/减码率 | + / − |
| 退出(控制→全屏→设置面板逐层退) | Esc |

## 设置面板(点齿轮 / 按 S)

可调:**分辨率 / 码率 / 帧率 / 编码(H.264 / H.265) / 模式 / 操控热键 / USB 网络共享开关**,改完点**应用**。

> 注意:**只有"码率"是热更新**(立即生效);其他改动需点「应用」或重开投屏才生效。

## 「控制」模式(点对点操控)

- 打开"控制"后,**画面即操控区**:光标自动锁进画面范围,鼠标左右键直接注入触控。
- **默认没有热键**,需在设置面板里自己绑一个开关热键。
- **退出只能用键盘**:`Esc` 或你绑的操控热键。鼠标点击退不出来,这是**防误触机制**。

## 投屏卡顿怎么办

投屏卡不影响触摸。想更流畅:

1. 设置面板**降分辨率 + 码率**。
2. 编码选 **H.265**(同画质更省带宽)。
3. 电脑和手机都靠近路由、走 **5GHz** WiFi。
4. USB 网络共享模式下会自动启用更稳的视频传输,延迟更低。

## 兼容性

- 适配 Android 11 ~ 16(高版本权限模型自动适配)。
- 60Hz 屏端到端延迟约 16–30ms,USB 网络共享组合下稳定居于低位。
